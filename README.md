# YPredictor — предиктор гаплогруппы Y-DNA и STR-матчер

**Бесплатный онлайн-инструмент для ДНК-генеалогии: предсказывает гаплогруппу Y-хромосомы по Y-STR-маркерам и находит ближайших совпаденцев в справочном корпусе.**

### 🔗 Открыть: **[ypredictor.github.io](https://ypredictor.github.io/)**

Вставьте STR-маркеры из теста FamilyTreeDNA — получите предсказанную ветвь Y-дерева и список ближайших родственных гаплотипов с оценкой возраста общего предка (TMRCA). Всё считается **прямо в браузере**: ни регистрации, ни загрузки данных на сервер.

---

## Зачем это нужно

По неглубоким тестам (37–67 маркеров) лаборатории часто останавливаются на широкой группе — например, J1-Z1842, под которую подходит значительная часть Кавказа. Чтобы узнать конкретный молодой субклад, обычно нужно заказывать дорогой Big Y. YPredictor оценивает эту глубокую ветвь **по вашим уже имеющимся STR-маркерам** — бесплатно и за секунды.

---

## Возможности

### 🧬 Предсказание гаплогруппы
- Спуск по филогенетическому Y-дереву **голосованием ближайших соседей** — каждый сосед голосует за ветвь, вес голоса тем больше, чем ближе гаплотип.
- **Clade-anchored режим** для откалиброванных клад (J1-Z1842, J2-M172, R1b-M269, R1a-M417, G2-P15): если ближайший сосед принадлежит известной кладе, предсказание строится только по её членам с параметрами, подобранными под эти популяции — заметно точнее.
- **Не только «кавказские» клады.** Инструмент принимает любой Y-STR-гаплотип и предсказывает по всему Y-дереву — в корпусе есть киты и более редких гаплогрупп (**T, N, O, E, L, Q, I** и другие), так что для них предиктор тоже находит соседей и определяет ветвь (обычным предсказанием, без клада-калибровки).
- Для каждого результата показываются **глубина** (сколько узлов дерева пройдено), **уверенность** (доля голосов за ветвь) и эвристика надёжности.
- Ссылки на терминальный SNP в FTDNA Discover прямо из результата.

> **Пример.** По неглубокому тесту лаборатория выдаёт общую группу J1. Предиктор по маркерам достраивает путь до молодой ветви: `J1 → ZS3042 → ZS4880 → B233` (B233 — одна из крупных даргинских ветвей). Такие ветви развились уже на Кавказе и в лабораторном результате обычно не видны.

### 🔍 STR-матчер
- Не только предикт: полноценный поиск **ближайших совпаденцев** — до 50 соседей, отсортированных по **возрасту TMRCA** (точка вероятности 0.95), а не просто по генетической дистанции.
- Для каждого соседа: гаплогруппа, проект, TMRCA в годах, `diff/cmp` (различия / сравненные маркеры), число маркеров.
- Корректный расчёт дистанции с учётом палиндромных маркеров (CDY, DYF395 и др.) и коррекции DYS389.

### ⌨️ Ввод
- Y-STR-маркеры в стандартном порядке FamilyTreeDNA: **Y-12 / Y-37 / Y-67 / Y-111** и STR-значения из **Big Y**.
- Значения из YSEQ и других провайдеров — вручную, если сохранён тот же порядок маркеров.
- Или просто **Kit-номер** из корпуса — инструмент подставит его маркеры.
- Разделители — пробел, запятая или перенос строки; палиндромы вида `15-17`.

### 📲 PWA — устанавливается как приложение
- **Работает офлайн**: service worker кеширует оболочку приложения и корпус (stale-while-revalidate), после первого визита предиктор доступен без сети.
- **Устанавливается на домашний экран** (Android/Chrome — по кнопке, iOS — «Поделиться» → «На экран „Домой“») и запускается как нативное приложение в отдельном окне.
- Тёмная/светлая тема, адаптивная вёрстка, быстрый старт.

### 🔄 Автообновление по расписанию
- Корпус и код **пересобираются по крону**: скрейп публичных проектов FTDNA → пересчёт соседей → сборка новой версии PWA → автоматический коммит и публикация.
- Версия — контент-хеш корпуса и кода; при изменении service worker подтягивает новые данные в фоне и показывает установленным пользователям тост **«Обновлено до версии …»** с предложением перезагрузить.
- Никаких ручных обновлений: открытая вкладка и установленное приложение сами получают свежий корпус.

### 🌐 Двуязычие и приватность
- Интерфейс **RU/EN** — автоопределение по языку браузера, переключатель вручную.
- **Маркеры не покидают страницу** — весь расчёт локальный, серверного бэкенда нет.
- В корпусе нет персональных имён: только гаплогруппа, маркеры и метка проекта.

### 🔗 Шеринг
- Кнопка «Поделиться» формирует ссылку с зашитыми маркерами — получатель открывает предиктор с уже готовым результатом.
- Текст шеринга содержит предсказанную гаплогруппу и топ ближайших совпаденцев (гаплогруппа · проект · возраст TMRCA).

## Как пользоваться

1. Откройте **[ypredictor.github.io](https://ypredictor.github.io/)** (или установите как приложение).
2. Вставьте свои Y-STR-маркеры из личного кабинета FamilyTreeDNA (или Kit-номер из корпуса).
3. Нажмите **«Предсказать»** — получите гаплогруппу, метрики надёжности и таблицу ближайших соседей.
4. Нажмите **📋** у соседа, чтобы подставить его маркеры, или **🔗**, чтобы поделиться результатом.

> Значок **?** рядом с каждой метрикой открывает пояснение — что означают «глубина», «уверенность», «режим» и «надёжно?».

## Точность и ограничения

Предсказание **вероятностное**. Глубокие, хорошо представленные в корпусе ветви оцениваются надёжно; неглубокие, редкие или пограничные — менее уверенно, это неотъемлемое свойство оценки по STR.

**Предсказанная гаплогруппа — гипотеза, а не подтверждённый результат.** Точную терминальную гаплогруппу подтверждает только SNP-тест (**FamilyTreeDNA Big Y** или адресный тест на отдельный SNP). Используйте предиктор, чтобы решить, какой тест заказывать.

Корпус и калибровка ориентированы прежде всего на популяции Кавказа (клады J1-Z1842, J2-M172 и др.), но инструмент принимает любые Y-STR-гаплотипы.

---

## Точность по кладам

Точность откалиброванных клад измеряется **leave-one-out кросс-валидацией** на китах с глубоким подтверждённым путём (≥5 узлов дерева) внутри клады: для каждого такого кита предсказание строится по его соседям (исключая его самого) и сравнивается с известной ветвью.

| Клада | Проверено китов | Верная ветвь | Ср. глубина совпадения |
|-------|:---:|:---:|:---:|
| **J1-Z1842** | 297 | 95.3 % | 3.28 |
| **J2-M172** | 1609 | 96.2 % | 9.39 |
| **R1b-M269** | 91 | 94.5 % | 5.98 |
| **R1a-M417** | 79 | 96.2 % | 5.08 |
| **G2-P15** | 60 | 95.0 % | 8.38 |

- **Верная ветвь** — доля предсказаний, где путь не ушёл на неправильный форк дерева (может быть короче или глубже истинного, но на той же линии).
- **Ср. глубина совпадения** — среднее число узлов пути (от корня), совпавших с истиной; чем больше, тем детальнее предсказание.

Числа — снимок текущего корпуса (2026-07) и пересчитываются при его обновлении. Инструмент не ограничен этими пятью кладами: он принимает любой Y-STR-гаплотип и справляется и с более редкими гаплогруппами — **T, N, O, E, L, Q, I** и другими — обычным (не клада-калиброванным) предсказанием.

## Как это устроено (технически)

- **Всё считается в браузере.** Математика (дистанция, TMRCA, предсказание) портирована в JS; дерево Y-хромосомы на клиенте не нужно — предрасчитанные пути (breadcrumbs) и метки клад вшиты в корпус.
- **Генетическая дистанция** — нормализованная по STR, с поаллельным учётом палиндромных маркеров (CDY, DYF395…) и коррекцией DYS389.
- **TMRCA** — оценка времени до общего предка методом GMRCA с экстраполяцией для далёких пар; соседи сортируются именно по возрасту, а не по «сырой» дистанции.
- **Алгоритм предсказания** — плюрализм-спуск по дереву: на каждом узле выбирается ветвь с наибольшим весом голосов соседей; спуск продолжается, пока ветви хватает поддержки и превосходства над второй (пороги `minSupport` / `margin` / `topK`). Под откалиброванной кладой голосуют только её члены с параметрами клады (clade-anchored).
- **Голосующие и не голосующие соседи.** За ветвь голосуют только **ближайшие** соседи (топ-K), вес голоса тем больше, чем ближе гаплотип; дальние соседи в списке видны, но за ветвь не голосуют. Часть корпуса помечена бейджем **«predictor only»** — это справочные киты из проектов-доноров: они добавляют голоса и расширяют охват, но не считаются полноценными совпаденцами. Отдельные справочные группы намеренно **исключены из голосования** (методологическое решение, чтобы не смещать оценку), хотя и остаются видимыми среди соседей. Действуют и доменные правила — например, при «голом» широком результате несовместимая под-ветвь не голосует. Слишком короткие гаплотипы (менее ~14 маркеров) не участвуют как совпаденцы: дистанция до них слишком шумная.
- **Параметры калибровки** (minSupport / margin / topK): J1-Z1842 `5 / 2.5 / 30` · J2-M172 `5 / 4 / 15` · R1b-M269 `5 / 2.5 / 10` · R1a-M417 `5 / 4 / 20` · G2-P15 `4 / 2.5 / 10`.
- **Корпус** — более 10 000 китов (гаплогруппа + маркеры + предрасчитанный путь), сжат gzip и грузится отдельным файлом; в нём нет персональных имён, только метка проекта.
- **PWA** — service worker кеширует оболочку и корпус по схеме stale-while-revalidate; версия сборки = контент-хеш данных и кода, обновление подтягивается в фоне.

## Источники

Названия гаплогрупп и топология Y-дерева следуют устоявшимся филогенетическим источникам:

- [YFull Y-tree (YTree)](https://www.yfull.com/tree/) — SNP-дерево Y-хромосомы с оценками возраста ветвей
- [ISOGG Y-DNA Haplogroup Tree](https://isogg.org/tree/) — консенсусное дерево ISOGG
- [FamilyTreeDNA Discover](https://discover.familytreedna.com/) — публичное гаплодерево и STR-ресурсы

## Об этом репозитории

Содержимое сайта (`index.html`, `corpus.gz`, `sw.js`, `manifest.webmanifest`, `sitemap.xml` и др.) — **артефакты автоматической сборки**. Они генерируются и публикуются кроном из основного движка расчётов (скрейп FTDNA → нормализация → предрасчёт соседей → сборка PWA) и обновляются при каждом изменении данных. Вручную правится только этот README.
